## 問題

N 個の整数 A0​,A1​,…,AN−1​ と M 個の整数 B0​,B1​,…,BM−1​ と整数 K が与えられます。

次の条件を満たす組 (i,j) の個数を答えるプログラムを作成してください。

- i は 0 以上 N−1 以下の整数
- j は 0 以上 M−1 以下の整数
- Ai​+Bj​=K

## 入力

入力は次の形式で与えられます。

```text
N M K
A0 A1 … AN−1
B0 B1 … BM−1
```

また、入力される値は次の制約を満たします。

- N,M は 1 以上 100 以下の整数
- K は 1 以上 200 以下の整数
- A0​,A1​,…,AN−1​ は 1 以上 100 以下の整数
- B0​,B1​,…,BM−1​ は 1 以上 100 以下の整数

## 出力

答えを出力してください。

## 入出力例

### 入力例 1

```text
3 3 8
5 3 7
1 1 5
```

### 出力例 1

```text
3
```

条件を満たす組は (i,j) = (1,2), (2,0), (2,1) の 3 つです。

### 入力例 2

```text
3 3 4
1 4 2
8 5 7
```

### 出力例 2

```text
0
```
