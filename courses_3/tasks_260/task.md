## 問題文

N 個の文字列 S0​,S1​,…,SN−1​ が与えられます。

これらの N 個の文字列の中に同じ 2 つの文字列があるかどうかを判定するプログラムを作成してください。

## 入力

入力は次の形式で与えられます。

```text
N
S0
S1
⋮⋮
SN−1
```

また、入力される値は次の制約を満たします。

- N は 1 以上 100 以下の整数
- Si​ は英小文字からなる長さ 1 以上 100 の文字列 (0≤i≤N−1)

## 出力

同じ文字列が含まれていれば Yes を出力し、含まれていなければ No を出力してください。

## 入出力例

### 入力例 1

```text
3
algo
algo
method
```

### 出力例 1

```text
Yes
```

S0​ と S1​ が同じ文字列です。
