## 問題

N 個の整数 A0​,A1​,…,AN−1​ が与えられます。

N 個の整数の最大値を求めるプログラムを作成してください。

## 入力

入力は次の形式で与えられます。

```text
N
A0 A1 … AN−1A
```

また、入力される値は次の制約を満たします。

- N は 1 以上 100 以下の整数
- Ai​ は −100 以上 100 以下の整数 (0≤i≤N−1)

## 出力

N 個の整数の最大値を出力してください。

## 入出力例

### 入力例 1

```text
5
-1 3 5 -7 -9
```

### 出力例 1

```text
5
```

−1,3,5,−7,−9 の中の最大値は 5 です。
